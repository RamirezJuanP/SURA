Gestión de Bases de Datos

Aprende a estructurar, administrar y optimizar bases de datos relacionales para almacenar y manejar información de forma eficiente y segura.

Contenido del curso

  • 1️⃣ Fundamentos de bases de datos
  • 2️⃣ Modelo relacional y diagramas ER
  • 3️⃣ Lenguaje SQL: consultas básicas y avanzadas
  • 4️⃣ Normalización y optimización de datos
  • 5️⃣ Introducción a bases de datos no relacionales (NoSQL)

Instructor

Andrés Herrera

Ingeniero de sistemas y especialista en bases de datos con más de 8 años de experiencia trabajando con MySQL, PostgreSQL y MongoDB.

Requisitos

Conocimientos básicos de lógica de programación y manejo de datos. Se recomienda haber completado el curso de Gestión Empresarial.
